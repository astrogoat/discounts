## Installation

You can install the package via composer:

```bash
composer require astrogoat/discounts
```

## Usage

```php
$discounts = new Astrogoat\Discounts();
echo $discounts->echoPhrase('Hello, Astrogoat!');
```

## Testing

```bash
composer test
```
